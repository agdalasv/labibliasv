## Características

- 📖 Múltiples traducciones (RVR, King James, Study Bible)
- 🌐 Español e Inglés
- 🔍 Búsqueda de versículos
- 📅 Planes de lectura
- 🔊 Text-to-Speech
- ⭐ Favoritos, notas y resaltado
- 🌙 Modo oscuro
